# 발전기금 전자 기부약정서 상세서

## 범위

이 프로그램은 태블릿에서 기부약정서를 작성하고, 전자서명을 받아, 기존 약정서 PDF 템플릿에 입력값과 서명 이미지를 합성해 PDF 파일을 생성한다.

## 포함 기능

- 약정서 입력 폼
  - 작성일
  - 기부자명 또는 기관명
  - 구분
  - 연락처
  - 주소
  - 기부금액
  - 기부용도
  - 납부방법
- 전자서명
  - 손가락, 스타일러스 펜 입력
  - 다시쓰기
  - 투명 PNG 서명 이미지 생성
- PDF 생성
  - `storage/templates/pledge/pledge-template.pdf` 배경 사용
  - `storage/templates/pledge/pdf-field-map.json` 좌표표 사용
  - 확정 입력 필드만 PDF에 표시
  - 보류 필드는 원본 약정서 확인 전까지 PDF 템플릿에서도 표시하지 않음
  - 생성 파일명: `YYYYMMDD_기부자명_금액.pdf`
  - 생성 위치: `storage/pledges/YYYY/MM/`
  - 완료 화면에서 생성 직후 PDF 다운로드 제공

## 제외 기능

- 관리자 로그인
- 관리자 약정서 목록, 검색, 상세, 메모
- Supabase Auth, Database, Storage
- PWA 서비스 워커, 앱 설치, 오프라인 캐시
- IndexedDB 임시저장 및 복구
- 결제 연동
- 외부 기부관리시스템 연동

## 보류 필드

현재 샘플 PDF 템플릿에는 있으나, 사용자가 확정한 2-1 요구 필드에는 없는 항목이다. 원본 HWP 약정서 확인 전까지 코드에서는 기본값을 유지하고 화면에는 노출하지 않는다.

- 생년월일/사업자번호
- 이메일
- 우편번호
- 납부 유형
- 납부 시작일
- 약정 개월
- CMS 출금일
- 은행명
- 계좌번호
- 예금주
- 개인정보 동의
- 기부금 영수증 신청

## 라우트

| 경로 | 목적 |
|---|---|
| `/` | 약정서 입력, 서명, PDF 생성 |
| `/complete/$pledgeId` | 생성 완료 결과와 다운로드 |

## API

| 경로 | 메서드 | 목적 |
|---|---|---|
| `/api/pledges` | `POST` | 입력값과 서명 이미지로 PDF 생성 |
| `/api/pledges/:id/pdf` | `GET` | 생성된 PDF 다운로드 |

## 파일 저장 규칙

```txt
storage/
  pledges/
    .index/
      {submissionId}.json
    2026/
      06/
        20260629_홍길동_10000000.pdf
  templates/
    pledge/
      pledge-template.pdf
      pdf-field-map.json
```

`submissionId`는 생성 결과 ID로 사용한다. 같은 `submissionId`가 다시 제출되면 기존 PDF를 반환한다.

## 필수 수령물

- 원본 HWP 또는 원본에서 변환한 PDF
- 실제 PDF 좌표표
- 기관명과 로고

원본 HWP가 없으면 원본 약정서와 동일한 레이아웃인지 확정할 수 없다.
